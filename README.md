# NeoHub Add-ons for Home Assistant

## Installation

1. In Home Assistant go to **Settings > Add-ons > Add-on Store**
2. Click the three-dot menu (top right) and select **Repositories**
3. Add this repository URL: `TODO`
4. Find **NeoHub** in the add-on store and click **Install**
